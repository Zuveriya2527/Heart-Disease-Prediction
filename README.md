# ❤️ Heart Disease Analysis & Risk Prediction System


🚀 **Live Application:**  
👉 https://heart-disease-prediction-6xaq.onrender.com

A modern, interactive web-based Heart Disease Risk Prediction system built using **Flask (Python)**.  
This application analyzes user health parameters and predicts heart disease risk levels based on intelligent risk scoring logic.

---

## 🚀 Project Overview

Heart Disease Analysis is a full-stack web application designed to:

- Predict heart disease risk
- Calculate BMI automatically
- Analyze lifestyle factors
- Provide categorized risk levels (Low, Moderate, High)
- Deliver a clean and user-friendly interface

This project demonstrates practical implementation of:
- Flask Web Framework
- Python Risk Modeling Logic
- HTML, CSS, Bootstrap Frontend
- Modular Project Architecture
- Cloud Deployment using Gunicorn

## 🎯 Risk Prediction Logic

The system calculates a risk score based on:

- Age group classification
- Body Mass Index (BMI)
- Smoking frequency
- Exercise habits
- Stress levels
- Gender risk factors

Risk Categories:
- 🔵 Low Risk
- 🟡 Moderate Risk
- 🔴 High Risk

##🟢EXAMPLES
Example 1 — Low Risk

Use these values:

Age: 25
BMI: 21
Blood Pressure: 110
Cholesterol: 170
Smoking: No
Diabetes: No
Physical Activity: High

Expected result:

Low Risk of Heart Disease

Reason:
Young age, healthy BMI, good activity, no smoking.

🟡 Example 2 — Moderate Risk
Age: 45
BMI: 26
Blood Pressure: 130
Cholesterol: 210
Smoking: No
Diabetes: No
Physical Activity: Medium

Expected result:

Moderate Risk

Reason:
Middle age + slightly higher cholesterol.

🔴 Example 3 — High Risk

BMI: 32
Blood Pressure: 150

High Risk of Heart Disease

Multiple strong risk factors.
